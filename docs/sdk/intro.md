# Mobile SDK Introduction

Welcome to the Appfastfly React Native SDK! 

The Appfastfly Mobile SDK enables seamless deep linking, deferred deep linking, and precise device fingerprinting for your React Native applications. Whether your users have your app installed or not, Appfastfly guarantees they will be taken to the exact content they clicked on.

## Key Features

- **Deferred Deep Linking**: If a user clicks your link but does not have the app installed, they are routed to the App Store or Google Play. Once they install and launch it, Appfastfly matches them and passes the original data payload directly into your app.
- **Universal Links & App Links**: Out-of-the-box support for iOS Universal Links and Android App Links to directly open your app.
- **Accurate Attribution**: We use advanced proprietary methods to securely and accurately attribute deferred installs, boasting industry-leading matching confidence.

## Supported Platforms

Currently, the Appfastfly SDK supports:
- React Native (iOS & Android)
- Built for the **New Architecture** (TurboModules and Codegen enabled)
